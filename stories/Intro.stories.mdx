# Get started

Design System is a library of reusable components that helps employees build user interfaces faster. The goal is to make the construction of durable UIs more productive and satisfying.

### **Install**

`npm install --save ...`

Or

`yarn add ...`


### **Use**

Import the module you want in its respective module

`import { ButtonModule } from 'package';`

And import it

`imports: [
  ButtonModule
]`

and use them like so

```
  <storybook-button color="blue-100" label="Button" />
```
